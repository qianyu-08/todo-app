
# ToDo App

シンプルな ToDo 管理アプリです。  
HTML / CSS / JavaScript を 1 ファイルにまとめているので、GitHub Pages でそのまま公開できます。

## デモ
https://qian-08.github.io/todo-app

## 機能

- タスク追加
- 完了 / 未完了の切り替え
- タスク編集
- タスク削除
- 完了タスクのみ削除
- 全削除
- フィルター表示
- localStorage によるデータ保存

## ファイル構成

```txt
todo-app/
├─ index.html
└─ README.md
````

## 使い方

1. リポジトリをクローン
2. `index.html` をブラウザで開く
3. タスクを追加して使う

## GitHub Pages で公開する手順

1. GitHub に新規リポジトリを作成
2. `index.html` をリポジトリのルートに配置
3. `README.md` を追加
4. GitHub の Settings → Pages を開く
5. Branch を `main`、Folder を `/root` に設定
6. Save する
